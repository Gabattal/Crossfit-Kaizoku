# CrossFit-Kaizoku

Booking app for a Gym.

## Technologies used:

* Vue.js
* Vuetify
* Firebase
* Cordova
* Android Studio

## Features:

* Inscription / Connexion
* Book courses with a waiting list system
* Create and modify courses (admin)
* Ban members (admin)
* Send a message to all members (admin)
* See who cancel his courses and when (admin)
